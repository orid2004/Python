## How to Play
```Right arrow:```  Step Right

```Left arrow:```  Step Left

```Down arrow:```  Step Down

```Z:```  Rotate

```Space:```  Drop

```Enter:``` Start Game

## Scoring
Each piece drop rewards with 5 points.  
Each row crash is equivalent to 15 piece drops, i.e., rewards with 75 points.
